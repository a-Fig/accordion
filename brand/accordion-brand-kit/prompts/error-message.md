# Task starter — Error / empty / loading states

Use the Accordion brand voice (see `system-prompt.md`). Write microcopy for a system state.

Rules
- Warm, not robotic. Specific, not vague. Calm, never alarmed.
- Say what happened, reassure that **nothing was lost** when true, and give the next step.
- One or two short sentences. No blame, no jargon dumps, no "Oops!".
- Reinforce the core promise where honest: folds are saved; data is recoverable.

Reference
- Error: "Couldn't reach the session. Your folds are saved — nothing was lost. Retry when ready."
- Empty: "No folded blocks yet. When context gets long, Accordion folds the oldest first — you'll see it here."
- Loading: "Rebuilding the context map…"

Task: [WHICH STATE + WHAT WENT WRONG/IS HAPPENING — fill in]
