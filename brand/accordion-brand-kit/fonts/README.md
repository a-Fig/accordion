# Fonts

Accordion uses two typefaces, both **SIL Open Font License (OFL)** — free to use,
embed, and redistribute.

| Role | Family | Weights used |
|---|---|---|
| Display + body | **IBM Plex Sans** | 400–700 (variable font; IBMPlexSans-Variable.ttf covers all) |
| Data / metrics | **IBM Plex Mono** | Regular 400, Medium 500 |

## Install

- **Local:** drop the `.ttf` files in this folder into your OS font book, or self-host them with `@font-face`.
- **Web (Google Fonts):**
  `https://fonts.googleapis.com/css2?family=IBM+Plex+Sans:wght@400;500;600;700&family=IBM+Plex+Mono:wght@400;500&display=swap`
